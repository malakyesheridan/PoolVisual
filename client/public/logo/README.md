# EasyFlow Studio Logo

Please add the EasyFlow Studio logo image files to this directory:

- `easyflow-logo.png` - Full logo (icon + text) - recommended size: 200x60px or larger
- `easyflow-logo-icon.png` - Icon only - recommended size: 100x100px or larger

The Logo component will automatically use these images when available, with fallback to SVG if images are not found.
